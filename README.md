## ✨ 特性

- 可靠的代码健壮：使用 Typescript 构建，提供完善的类型定义文件
- 完善的文档能力：支持文档记录，支持 demo 演示
- 完整的测试用例：配套完整的测试用例，帮助您提升项目健壮性

## 📦 安装

```bash
$ npm i code-hooks -S
# or
$ pnpm add code-hooks
#or
$ yarn add code-hooks
```

## 🔨 使用

```ts
import { useMount } from 'code-hooks';
```
